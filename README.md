# Airbnb Power BI Dashboard
This project analyzes Airbnb listings, pricing trends, and city-level performance using Power BI.

 Key Insights
- Top cities contribute a major share of listings and reviews
- Listings peaked around 2015 and declined after
- Entire place listings have higher average prices

 Tools Used
- Power BI
- Data Visualization
